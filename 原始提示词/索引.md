## 短片（多分镜剪辑型）

- **[《丧尸清道夫》](./丧尸清道夫.md)** ⭐ — 让 PJ Ace 称为"近年来最佳短片之一"的作品。
  - 镜头一：鸵鸟相遇（7 分镜完整版）
  - 镜头二：MJ 舞蹈段（完整）
  - 镜头三：剃头/踢丧尸头（仅 2 分镜片段，从直播转录拼出）
  - 直播提到但未公开：游轮钓鱼、餐厅生图、约会、日落 等（受 60 分钟长片授权 NDA 限制）

## 变身（单镜头一镜到底）

- **[假面骑士 / 特摄变身系列](./假面骑士-变身系列.md)** ⭐ — 5 个 IP 变体共享同一套模板：
  - 蟑螂侠战衣（HENSHIN, 暗红水晶）
  - 火焰恶魔战衣（火焰冠状犄角）
  - 雷龙战甲（AMAZONS, 暗蓝水晶）
  - 彩虹麋鹿战衣（雄性麋鹿角）
  - 紫鸢-女武神战甲（韩语「변신」, 暗紫水晶）
  - 紫鸢-女武神战甲（飞行版）

- **[《卡莎》LOL 变身](./卡莎-LOL变身.md)** — 三个版本：
  - 15 秒分镜版（5 个分镜）
  - 20 秒分镜版（4 个分段，多了恐惧铺垫）
  - 5 秒延长版（接 20 秒版的飞行轰炸）

## 重型机甲

- **[《环太平洋 + 高达 + 头号玩家》](./环太平洋-高达.md)** — 跳机 → 虚拟手套 → 数字方块组装机甲 → 砸海 → 海兽出现。8 段分镜，FPV 运镜。

## 风格模板（人物 + 场景骨架）

- **[《赛博江湖》](./赛博江湖.md)** — 邵氏 + 蒸汽朋克 + 武侠的画质骨架。东厂公公 / 机械义体 / 烛火庭院。**画面内容留空，自己写**。

## 战斗组合（武器充能 + 打斗）

- **[《合金装备》](./合金装备-武器充能与打斗.md)** — 武器充能段 + 打斗段。配套有场景 / 怪物 / 武器 / 动力装甲 4 张图的生成提示词。后期剪辑拼接。

## 怎么用

1. **新手**：从 [《丧尸清道夫》](./丧尸清道夫.md) 开始读，对比 [方法论.md](../方法论.md) 看模板如何落地。
2. **想做变身**：[假面骑士系列](./假面骑士-变身系列.md) 一次看完 5 个变体，比看一遍模板更直观地理解"哪些是变量、哪些是骨架"。
3. **想做打斗 / 长片**：[卡莎](./卡莎-LOL变身.md) + [合金装备](./合金装备-武器充能与打斗.md) 演示了"分段生成 + 剪辑拼接"的标准流程。
4. **想做静态氛围 / 海报**：[赛博江湖](./赛博江湖.md) 是个干净的画质模板。

## 关于版权词

原作里包含一些 IP 名（仮面ライダー BLACK SUN / AMAZONS / 卡莎 / 环太平洋 / 高达 / 合金装备 等）。
**Seedance 2.0 对版权词比较敏感**，照搬可能被拦。原作者建议：
> 替换提示词文案中涉及影视作品的字眼和词语，或删除部分字眼/标点，不改变大致意思就行。

我们在 [模板/](../模板/) 目录里提供了**去掉 IP 的通用骨架**，可以直接套用。
