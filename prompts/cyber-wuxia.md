# 《赛博江湖》提示词模板（原文）

---

## 提示词

### 核心氛围与画质

邵氏电影风格，蒸汽朋克与古风武侠的完美结合，80 年代的技术彩色武术电影风格，复古港风，机械武侠，史诗构图，深色阴影，柯达 35mm 复古胶片拍摄，跳过漂白胶片质感，画面高光部位添加柔焦效果，画面质感丰富，真人实景拍摄，写实电影感。

**声音**：只保留同期声，不要背景音乐。

### 人物

一位年轻的东厂公公（明朝太监），面容精致，眼神锐利而坚定，带有一丝冷艳。他身着传统的深红色立领官服，头戴官帽，衣料上有四爪金龙刺绣，尽显古典雅致。最引人注目的是他右脸颊及耳部覆盖着一套精密复杂的银灰色机械义体，义体表面有精密的齿轮、管线和金属刻痕，与他细腻的皮肤形成强烈的质感对比。

### 场景氛围

场景设定在一个夜晚的中式古典庭院，背景是模糊的飞檐斗拱和青瓦白墙，烛火摇曳，营造出一种神秘而压抑的氛围。空气中弥漫着薄雾，在他的皮肤和义体上凝结成晶莹的水珠，反射着冷冽的光线。

### 构图

采用中近景特写，人物占据画面主体，视线略微偏向画面右侧，形成一种动态的张力。背景中的人物被虚化处理，作为环境的一部分，突出了主角的存在感。

### 色彩与光影

非自然戏剧化人造光。主色调为冷色调，以青灰、银白和暗红色为主，烛火发出的暖光映衬着面部。烛光暖光来自侧前方，月光的冷光来自侧背方，光源与人物呈"三点一线"的布置。光线是带有电影感的硬光，在他的面部和义体上形成了清晰的明暗对比，突出了皮肤的光泽和金属的冰冷质感。

### 画面风格

邵氏电影风格，融合了中国古典美学与蒸汽朋克元素，呈现出一种超现实的未来主义复古感。画面质感追求 80 年代香港武侠电影的胶片颗粒感，同时又具备现代商业摄影的高清细节，色彩浓郁，对比度高，充满戏剧张力。

### 动态描述

他微微侧头，目光如炬，阴柔狠辣，仿佛正准备迎接一场即将到来的对决。

## 学习要点

1. **没有时间分镜 / 没有分镜列表** —— 因为这是一个**人物 + 场景设定模板**，时长由你自己定。
2. **三点光源布置写法** —— "烛光暖光来自侧前方，月光的冷光来自侧背方，光源与人物呈三点一线的布置"。**用真实摄影术语描述光线**，比"暖光冷光对比"信息密度高。
3. **质感对比是核心**：精致皮肤 vs 机械义体 / 青灰冷色 vs 暗红官服 / 古典庭院 vs 蒸汽朋克 —— 三层对比叠加。
4. **可替换变量**：把"东厂公公 + 庭院 + 机械义体"换成"侠客 + 客栈 + 仿生眼"或"道姑 + 道观 + 机械臂"，这套画质骨架照样可用。

## 应用建议

这个模板适合做**人物定妆照 / 场景宣传图 / 单镜头氛围片**。
如果要做完整短片，配合方法论里的"分镜表"（按写法 B：景别 + 构图 + 运镜手法 + 画面内容）补全分镜段。
