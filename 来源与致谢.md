## 仓库做了什么

我们**没有**做的：
- 没有改写、扩充或"再创作" Mx-Shell 的原始提示词
- 没有去掉他署名 / 冒充作者

我们**做了**的：
- 从直播 + 文档里整理出 [方法论.md](./方法论.md)（系统化讲解他的 5 段式结构）
- 把分散的 17 条 FAQ + 直播 Q&A 整合成 [实战FAQ.md](./实战FAQ.md)
- 把 doc/docx 转换并清理为分类的 [原始提示词/](./原始提示词/) 文件
- 抽出去 IP 的通用 [模板/](./模板/)
- 写了一个 [Claude Code Skill](./.claude/skills/shortfilm-prompt/)（`/shortfilm-prompt`），让 Claude 能按这套方法生成提示词

每一份原始提示词的开头都标注了来源文档和日期。
